Tech Stack

Backend
FastAPI — web framework
SQLAlchemy — ORM
Alembic — migrations
APScheduler — background jobs
httpx — async HTTP (PMS calls)
python-jose — JWT auth
Frontend
Jinja2 — portal templates
HTMX — dynamic updates
Alpine.js — lightweight reactivity
Tailwind CSS — styling
Chart.js — analytics charts
—
Infrastructure
PostgreSQL — database
Redis — session cache
dnsmasq — DHCP + DNS
iptables/nftables — network
Nginx — reverse proxy
Uvicorn — ASGI server
Testing
pytest + pytest-asyncio — unit & integration
httpx TestClient — API testing
Mock PMS adapters — ไม่ต้องมี PMS จริงตอนทดสอบ
pytest-cov — coverage report
Deployment (Ubuntu)
systemd service — auto-start FastAPI
docker-compose — PostgreSQL + Redis
iptables-persistent — rules survive reboot
logrotate — manage log files
